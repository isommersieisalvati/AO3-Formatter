# AO3 Formatter

## Features

-   Adjust line height;
-   Adjust font size;
-   Adjust max width;
-   Adjust indentation before paragraphs.

-   调整行间距；
-   调整字体大小；
-   调整文章所占屏幕最大宽度；
-   调整段前缩进。
